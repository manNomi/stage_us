# 개발자의 공부 방법
	- 주입식 교육 공부 X 
	- 탑다운 : 목표를 정하고 그에 맞는 과정을 설계하는것 
		- EX : 계산기 만들기 
			-> 일반인 : 계산기 만들기를 검색해본다
			-> 개발자 : 과정을 고민한다 
			    (버튼 9개 만든다 / 꾸민다 / 버튼을 눌렀을때 출력한다 / 버튼을 누르면 출력하는 모니터를 만든다)
				-> step by step 으로 간다 
				-> 과정을 검색해 공부한다 
	- 탑다운 공부 필요한 능력 
		- 논리와 근거 : 과정이 맞는지 계속 검증해야함 (떠먹는 습관 버리기)
		- 구글링 스킬 : 개발자에게 가장 중요한 기술 / 내가 원하는 자료를 바로 찾는 스킬 
			-> 찾은 자료를 정제하는 능력 : 옳은 글인지 확인
			-> 검색 결과를 내 코드에 녹여내는 능력 : 찾은 결과를 crtl c+ v 하지 말것
			-> 내가 원하는 정보가 잘 나오는 검색어를 찾는 능력 
			-> 끝까지 찾아보는 능력 : 흐름만 알려고 하지말고 응용할 수 있는 실력을 기를것
